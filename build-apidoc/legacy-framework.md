# Legacy .NET Framework Integrations

The classic/legacy API set includes Windows-only / .NET desktop framework components and integrations like MVC 5, Web API 2, WCF, SignalR, and related packages. While you may also be able to use some of the current APIs on Windows with the classic .NET framework, you won't be able to use these legacy integrations with newer .NET core projects.
